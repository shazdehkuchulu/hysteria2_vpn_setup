## فایل‌های راهنما

&#x202b;
### ۱. [راهنمای نصب Reality VPN برای کاربران مک/لینوکس](hysteria2-setup-guide.md)

&#x202b;
راهنمای کامل نصب سرور VPN شخصی با پروتکل Hysteria2 برای کاربرانی که از مک یا لینوکس استفاده می‌کنند:

&#x202b;
- ساخت کلید SSH
- تنظیم DNS
- نصب Hysteria2 با Let's Encrypt
- مدیریت کاربران
- عیب‌یابی

&#x202b;
**Hysteria2 چیست؟** یک پروتکل VPN سریع و مدرن که بر پایه UDP کار می‌کند و برای دور زدن سانسور طراحی شده است.

---

&#x202b;
### ۲. [راهنمای نصب Reality VPN برای کاربران ویندوز](hysteria2-setup-guide-windows.md)

&#x202b;
راهنمای کامل نصب سرور VPN شخصی با پروتکل Hysteria2 برای کاربرانی که از ویندوز استفاده می‌کنند:

&#x202b;
- ساخت کلید SSH با PowerShell و PuTTYgen
- اتصال به سرور با PowerShell و PuTTY
- تنظیم DNS
- نصب Hysteria2 با Let's Encrypt
- مدیریت کاربران
- عیب‌یابی

&#x202b;
### ۴. [راهنمای نصب Psiphon Conduit](conduit-setup-guide.md)

&#x202b;
راهنمای راه‌اندازی نود داوطلبانه Conduit برای کمک به کاربران در مناطق تحت سانسور:

&#x202b;
- دانلود و نصب باینری Conduit
- ساخت سرویس systemd برای اجرای خودکار
- تنظیم محدودیت پهنای باند
- روش جایگزین با Docker

&#x202b;
**Conduit چیست؟** 

&#x202b;
Conduit یک نود رله داوطلبانه برای شبکه [Psiphon](https://psiphon.ca/) است. با اجرای Conduit روی سرور خود، شما به عنوان یک پل ارتباطی عمل می‌کنید و به کاربرانی که در مناطق تحت سانسور هستند کمک می‌کنید تا به اینترنت آزاد دسترسی پیدا کنند.

&#x202b;
**تفاوت با VPN شخصی:**

&#x202b;
| ویژگی | VPN شخصی (Hysteria2) | Conduit |
|-------|---------------------|---------|
| استفاده | فقط برای خودتان و کاربرانتان | کمک به کاربران ناشناس شبکه Psiphon |
| مدیریت کاربران | شما کنترل کامل دارید | اتوماتیک توسط شبکه Psiphon |
| نوع کمک | دسترسی مستقیم | رله/پل ارتباطی |
| هدف | استفاده شخصی | کمک داوطلبانه به دیگران |
