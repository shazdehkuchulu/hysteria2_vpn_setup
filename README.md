# راهنمای نصب و راه‌اندازی Reality VPN با Hysteria2

&#x202b;
این مجموعه شامل راهنماهای کامل برای نصب و پیکربندی سرور VPN با پروتکل Hysteria2 است.

---

&#x202b;
## فایل‌های راهنما

&#x202b;
### ۱. [راهنمای نصب برای کاربران مک/لینوکس](hysteria2-setup-guide.md)

&#x202b;
راهنمای کامل نصب سرور VPN برای کاربرانی که از مک یا لینوکس استفاده می‌کنند:

&#x202b;
- ساخت کلید SSH
- تنظیم DNS
- نصب Hysteria2 با Let's Encrypt
- مدیریت کاربران
- عیب‌یابی

---

&#x202b;
### ۲. [راهنمای نصب برای کاربران ویندوز](hysteria2-setup-guide-windows.md)

&#x202b;
راهنمای کامل نصب سرور VPN برای کاربرانی که از ویندوز استفاده می‌کنند:

&#x202b;
- ساخت کلید SSH با PowerShell و PuTTYgen
- اتصال به سرور با PowerShell و PuTTY
- تنظیم DNS
- نصب Hysteria2 با Let's Encrypt
- مدیریت کاربران
- عیب‌یابی

---

&#x202b;
### ۳. [راهنمای فعال‌سازی ربات تلگرام](telegram-bot-setup-guide.md)

&#x202b;
راهنمای راه‌اندازی ربات تلگرام برای مدیریت کاربران VPN:

&#x202b;
- ساخت ربات با BotFather
- فعال‌سازی ربات روی سرور
- اضافه کردن چند ادمین
- غیرفعال کردن ربات
